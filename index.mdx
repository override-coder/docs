Poly Trader OpenAPI gives external developers and partners programmatic access to third-party Poly Trader API workflows. Every documented endpoint uses API Key + HMAC-SHA256 authentication, giving you secure, auditable access to wallets, orders, positions, copytrade, and AFK automated strategies.

## Get started in four steps

<Steps>
  <Step title="Obtain an API key">
    Contact your Poly Trader partner manager or create a delegated user API key through the Partner API. Store the key secret securely because it is shown only once.
  </Step>
  <Step title="Build your HMAC-SHA256 signature">
    Construct the canonical string from method, path, query, timestamp, nonce, and body hash, then sign it with your secret key using HMAC-SHA256.
  </Step>
  <Step title="Send your first authenticated request">
    Call `GET /wallet` with the four required headers: `X-API-Key`, `X-Timestamp`, `X-Nonce`, and `X-Signature`.
  </Step>
  <Step title="Explore the full API">
    Place orders, manage copytrade configs, create AFK strategies, and more using the complete API reference.
  </Step>
</Steps>
