## Get started in four steps

<Steps>
  <Step title="Obtain an API key">
    Contact PolyCop to receive an API key for your account. Store the secret immediately. It is shown only once.
  </Step>
  <Step title="Build your HMAC-SHA256 signature">
    Build the canonical string from the method, path, query, timestamp, nonce, and body hash. Then sign it with your secret key using HMAC-SHA256.
  </Step>
  <Step title="Send your first authenticated request">
    Call `GET /wallet` with the four required headers: `X-API-Key`, `X-Timestamp`, `X-Nonce`, and `X-Signature`.
  </Step>
  <Step title="Explore the full API">
    Use the API reference for request bodies, response schemas, and error responses.
  </Step>
</Steps>
